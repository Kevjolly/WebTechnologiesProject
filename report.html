Starting the website:

Before starting the server, Elasticsearch needs to be downloaded:

  • Download and unzip Elasticsearch (VERSION 6.6.1!) https://www.elastic.co/cn/downloads/past-releases/elasticsearch-6-6-1
  • Run bin/elasticsearch (or bin\elasticsearch.bat on Windows)
  • Now you can run the server (some node modules might be necessary). The website should now be available at localhost:8080.
Versions used: node v11.9.0, npm v6.5.0, elasticsearch v6-6-1.

Brief description of the website and work done:

TeamUp is a website where people can find projects to work in team. They can create a project and people can join those projects. They can also communicate with messages between each other. A large part of the server side has been done and the front-end was started. As we had already worked on projects with RDBMS SQL databases or frameworks like Bootstrap, we decided to try to use new frameworks like for instance: Materialize.css for the front-end or Elasticsearch for the database. Therefore, part of the time was spent learning about those frameworks.
